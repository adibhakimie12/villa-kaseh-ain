# Villa Kaseh Ain

Source code untuk website Villa Kaseh Ain.

## Structure

- `Homepage/` - website utama Vite + React
- `Booking/` - folder lama / reference
- `Contact/` - folder lama / reference

## Run Locally

```bash
npm install
npm run dev
```

## Build

```bash
npm run build
```

## Media Folder

Fail media masa depan telah disediakan di:

- `Homepage/public/media/videos/`
- `Homepage/public/media/images/gallery/`

Buat masa ini website guna placeholder semasa. Bila video dan gambar sebenar sudah siap, kita boleh tukar mapping ke folder ini.

## Go Live

Pilihan paling cepat:

1. Push repo ini ke GitHub
2. Import repo ke Vercel
3. Set Root Directory kepada `Homepage`
4. Deploy dan guna URL `.vercel.app`

Untuk domain sendiri:

- domain utama yang dibeli ialah `villakasehain.com`
- `www.villakasehain.com` cuma subdomain kepada domain utama itu
